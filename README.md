# sql-practica

Este repositorio contiene los entregables correspondientes a la práctica de Advanced SQL & Data Warehousing (DW) para KeepCoding. A continuación, se detallan los archivos incluidos:

1.  keepcoding_diagrama.pdf

Documento en formato PDF que contiene el Diagrama Entidad-Relación del modelo de base de datos de KeepCoding.
Este archivo corresponde al punto 1 de la práctica.

2.  keepcoding_schema_postgresql.sql

Script SQL que crea el modelo relacional derivado del diagrama anterior, diseñado para ejecutarse en PostgreSQL.

Este archivo corresponde al punto 2 de la práctica.

3.  ivr_practica_bigquery.sql

Script SQL que resuelve los ejercicios relacionados con el sistema IVR (Interactive Voice Response), desarrollado en BigQuery SQL.
Este archivo corresponde al punto 3 a 13 de la práctica.
